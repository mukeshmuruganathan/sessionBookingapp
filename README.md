# 📅 Session Booking Application

A full-stack MERN application for managing appointments and session bookings. Features a clean, modern UI for clients and a comprehensive dashboard for administrators.

## 🚀 Features

- **Client Side**: Easy-to-use booking form with real-time slot availability.
- **Admin Dashboard**: Overview of all bookings, statistics, search functionality, and cancellation management.
- **Database**: Connected to MongoDB Atlas for reliable data storage.
- **Validation**: Strict input validation to prevent double bookings and invalid data.

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS, Lucide Icons, Date-fns.
- **Backend**: Node.js, Express, Mongoose.
- **Database**: MongoDB Atlas.

## 📦 Getting Started

### Prerequisites

- Node.js installed.
- MongoDB Atlas cluster (Connection string already configured in `.env`).

### Installation

1. Clone the repository.
2. Install Backend dependencies:
   ```bash
   cd Backend
   npm install
   ```
3. Install Frontend dependencies:
   ```bash
   cd frontend/frontend
   npm install
   ```

### Running the App

- **Backend**: `npm start` (from the Backend folder).
- **Frontend**: `npm run dev` (from the frontend/frontend folder).
